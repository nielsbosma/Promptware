# Feedback Mode

When invoked with the `-Feedback` flag, the user is providing feedback about your behavior — not asking you to perform your normal task.

## Instructions

1. Read the feedback from Args (everything after `-Feedback`)
2. Read your current `Program.md`
3. Apply the feedback by updating `Program.md` with new or modified instructions
4. Read and update relevant Memory files if needed
5. Summarize what you changed

Do NOT execute your normal task. Only process the feedback and improve.
